Z-Transform

The zzz- transform for a discrete-time signal x[n]\text{x}[n]x[n] is given as:

X(z)=∑_n=−∞∞x[n] z−n \text{X}(\text{z}) = \sum\_{n=-\infty}^ {\infty} \text{x}[n]~ \text{z}^{-n} X(z)=∑_n=−∞∞x[n] z−n

where X(z)\text{X}(\text{z})X(z) is a complex valued function of complex variable z\text{z}z.

z\text{z}z transform always has two parts:

(i) Mathematical expression X(z)\text{X}(\text{z})X(z)

(ii) Region of convergence (ROC) - region in the complex z\text{z}z- plane where the sum X(z)\text{X}(\text{z})X(z) converges.

The region of convergence has radial symmetry. The ROC is always bounded by poles.

Examples

(i) Let x[n]\text{x}[n]x[n] be δ[n]={1      n=00otherwise\delta[n] = \left\{\begin{matrix} 1 \quad ~~~~~~n=0 \\ 0 \quad \text{otherwise} \end{matrix}\right.δ[n]={1      n=00otherwise​

The z\text{z}z- transform is given by,

X(z)=∑_n=−∞∞x[n] z−n\text{X}(\text{z}) = \sum\_{n=-\infty}^ {\infty} \text{x}[n]~ \text{z}^{-n} X(z)=∑_n=−∞∞x[n] z−n

         =∑_n=−∞∞δ[n] z−n~~~~~~~~~ = \sum\_{n=-\infty}^ {\infty} \delta[n]~ \text{z}^{-n}          =∑_n=−∞∞δ[n] z−n

         =1 ~~~~~~~~~= 1          =1

ROC of δ[n]\delta[n]δ[n] : complete zzz-plane

(ii) x[n]=δ[n−n0]\text{x}[n] = \delta[n-n_0]x[n]=δ[n−n0​], where n0>0n_0 > 0n0​>0 and an integer

The z\text{z}z- transform is given as:

X(z)=∑_n=−∞∞x[n] z−n\text{X}(\text{z}) = \sum\_{n=-\infty}^ {\infty} \text{x}[n]~ \text{z}^{-n} X(z)=∑_n=−∞∞x[n] z−n

         =∑_n=−∞∞δ[n−n0] z−n~~~~~~~~~ = \sum\_{n=-\infty}^ {\infty} \delta[n-n_0]~ \text{z}^{-n}          =∑_n=−∞∞δ[n−n0​] z−n

         =z−n0~~~~~~~~~ = \text{z}^{-n_0}         =z−n0​

         =(1z)n0~~~~~~~~~ = {\left ( \frac{1}{\text{z}} \right )}^{n_0}         =(z1​)n0​

ROC: whole z\text{z}z-plane except z=0\text{z} = 0z=0

(iii) x[n]=δ[n+n0]\text{x}[n] = \delta[n+n_0]x[n]=δ[n+n0​], where n0>0n_0 > 0n0​>0 and an integer

The z\text{z}z- transform is given as:

X(z)=∑_n=−∞∞x[n] z−n\text{X}(\text{z}) = \sum\_{n=-\infty}^ {\infty} \text{x}[n]~ \text{z}^{-n} X(z)=∑_n=−∞∞x[n] z−n

         =∑_n=−∞∞δ[n+n0] z−n~~~~~~~~~ = \sum\_{n=-\infty}^ {\infty} \delta[n+n_0]~ \text{z}^{-n}          =∑_n=−∞∞δ[n+n0​] z−n

         =zn0~~~~~~~~~ = \text{z}^{n_0}         =zn0​ using shifting property of impulse signals

ROC: whole z\text{z}z-plane except ∣z∣=∞|\text{z}|=\infty∣z∣=∞

Poles and zeros

We are interested in z\text{z}z-transforms which are in the form of ratio of polynomials in z\text{z}z.

X(z)=N(z)D(z)\text{X}(\text{z}) = \frac{N(\text{z})}{D(\text{z})} X(z)=D(z)N(z)​

Numerator N(z)=0\text{N}(\text{z}) = 0N(z)=0 provides zeros of X(z)\text{X}(\text{z})X(z)

X(z)=0 \text{X}(\text{z}) = 0 X(z)=0

Denominator D(z)=0\text{D}(\text{z}) = 0D(z)=0 provides poles of X(z)\text{X}(\text{z})X(z)

X(z)=∞ \text{X}(\text{z}) = \infty X(z)=∞

Note

  1. Poles play an important role in deciding the ROC, zeros do not. The ROC can not contain any poles.

  2. Different time domain signals can have same z\text{z}z-transform expression X(z)\text{X}(\text{z})X(z) but with different ROC.

  3. For the system to be causal, the ROC would be outside the outermost pole.

  4. For the system to be stable, the unit circle should be part of the ROC.

  5. For a system to be causal and stable, all the poles should be inside the unit circle.
